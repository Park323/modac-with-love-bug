# Map Selector

Transport Ship 2.0 맵 위에서 클릭+드래그로 웨이포인트 `x, y, rot`를 뽑아주는 툴입니다.

## 실행

```bash
python server.py
```

브라우저에서 `http://localhost:8080` 열기.

> `pip install` 불필요 — Python 표준 라이브러리만 사용합니다.

## 사용법

1. 맵 위에서 **클릭 + 드래그** — 드래그 방향 = 캐릭터가 바라볼 방향(rot)
2. 마우스 놓으면 웨이포인트 추가
3. 오른쪽 패널에서 **Copy JSON** 또는 **Download** 로 추출

## 출력 형식

```json
[
  {"x": 116.0, "y": 261.0, "rot": 90.0},
  {"x": 990.0, "y": 320.0, "rot": 45.0}
]
```

## 좌표 체계

- 원점: 좌측 상단 (0, 0)
- +x: 오른쪽, +y: 아래쪽
- rot 0° = 북쪽(위), 90° = 동쪽(오른쪽), 시계방향
- 맵 크기: 1980 × 654 픽셀
